Pseudo-class

Pseudo-classes are used to make your HTML elements more interactive. There are many different types of pseudo-classes, such as:

  • hover
  • focus
  • active
  • and many more . . .

The format below is used for pseudo-classes:

				
					selector:pseudo-class {

					    property:value;

					}

For the sake of example, let's create a button class.

				
					.button {

					    background-color: #fff;

					    color: black;

					    text-decoration: none;

					    padding: 4px 8px;

					    border: 1px solid #ccc;

					    border-radius: 4px;

					}
