The TechHOUNDS Signup Form

This is an article/tutorial on how to use the TechHOUNDS Signup Application Form. By the end of this you should know how to make changes and how to make them look pretty and stuff.

Some Background

The website (at the time of this writing) does not have its own MySQL, or an equivalent, database. TechHOUNDS also doesn't have a dedicated server computer or anything that data can be submitted to at any point in time. So, we need a different method to store the signup data. The easiest solution is by making the form in Google Forms.

How This Works

By using Google Forms, any data that gets submitted through that form is submitted to a Google Spreadsheet. We can then take this form and apply custom CSS and JS to it as per our choosing, while maintaining the same functionality. So by creating a Google Form, taking the form's HTML output and putting that HTML somewhere else (such as the TechHOUNDS website), it retains the functionality while also giving us room to customize it.
